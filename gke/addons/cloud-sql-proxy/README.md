# cloud-sql-proxy
**Category**: Database | **Source**: Sidecar | **Platform**: GKE
## Overview
Secure connectivity to Cloud SQL from GKE using the proxy sidecar pattern.
## Quick Start
```bash
./install.sh --namespace my-app --instance my-project:us-central1:my-db
```
