# EKS Node Groups

This directory contains definitions for various types of EKS node groups.

## Node Group Types

- **Managed Node Groups**: AWS-managed lifecycle.
- **Self-Managed Node Groups**: User-managed EC2 instances.
- **Fargate Profiles**: Serverless compute for pods.
- **Spot Node Groups**: Cost-optimized using AWS Spot Instances.
- **GPU Node Groups**: Specialized instances for ML/HPC.
