# eks-pod-identity-agent
**Category**: Identity | **Source**: EKS Addon | **Platform**: EKS
## Overview
Enables EKS Pod Identity for simplified IAM role assignment (alternative to IRSA).
## Quick Start
```bash
./install.sh --cluster my-cluster
```
