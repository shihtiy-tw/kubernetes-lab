# metrics-server
**Category**: Observability | **Source**: Helm | **Platforms**: All
## Overview
Provides resource metrics (CPU/memory) for HPA and `kubectl top`.
## Quick Start
```bash
./install.sh
./status.sh
kubectl top nodes
kubectl top pods
```
